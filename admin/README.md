# B-CashPay Admin Dashboard

Phase 3 placeholder. Will be a Next.js admin panel for:

- Payment link management (list, search, manual confirm)
- Bank account configuration (add, edit, enable/disable scraping)
- Scraper monitoring (status, logs, screenshots, manual trigger)
- Transaction history (deposits, matches, unmatched)
- API client management (create, rotate keys)
- Webhook log viewer
- Dashboard with key metrics

Tech stack (planned):
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- JWT authentication
